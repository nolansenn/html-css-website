About this site

The Scoop

I'm a junior at George Mason University in Fairfax, Virginia. Im currently pursuing a B.S. in Information Technology with a concentration in Cloud Computing. I created this website to showcase my journey to becoming a cloud engineer. The website is a work in progress and is coded with HTML, CSS, and Javascript. This is the first website I've created with CSS and JavaScript, which was certainly a learning curve. I found it particulary tricky to get the mobile version working correctly. I find myself fond of Web Development even though I don't consider it my main interest.

The deployment and management of this website is done through AWS. I utilize the free tier on AWS to pay a very low amount for this website. The most expensive cost was actually buying the domain. I use an S3 bucket to serve the static web files. I use Cloudfront to deliver the content and Certificate Manager for an SSL certificate. Finally, I use Route 53 to point towards my custom DNS domain. That's all it takes to host the website. To automate updating this website, I used AWS CodePipeline to sync the S3 bucket to a GitHub Repository that will update whenever I push new changes. I then created an event that invokes a AWS Lambda function to invalidate the Cloudfront cache whenever the S3 bucket is updated. I also have a budget setup and custom billing alerts. I'm currently working on a JavaScript website traffic counter that stores the count in a DynamoDB.
